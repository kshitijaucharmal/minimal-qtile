# Neo-Spark
![neospark](neospark.png)

## Installation
Just clone the repo and cp the files to ~/.config/nvim
```bash
git clone https://github.com/kshitijaucharmal/neospark.git
mkdir -p ~/.config/nvim
cp -r * ~/.config/nvim/
```

## Contents
+ Package Manager - Packer (lua)
+ Other settings (vimscript)
